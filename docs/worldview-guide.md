# 世界観ガイド

## 一文で言う

このゲームは、魔物のスライムを育てるゲームではない。
人間の国家が、兵士・工兵・参謀・兵站を一つの弾性体のように運用する「弾性軍制」を発明し、その軍制をどう成長させ、どこで戦わせ、どの国家と交渉するかを決める戦争シミュレーションである。

## 世界の核

舞台は、複数の人間国家が前線・資源地帯・旧要塞を奪い合う大陸である。
この世界の戦争は、個別部隊の数や英雄の強さだけでは勝てない。密集しすぎれば崩れ、広がりすぎれば割れ、命令が遅れれば戦線がねじれる。

そこで各国は、兵士を点ではなく「面」として扱う軍学を発展させた。隊列、信号、工兵線、補給、軍医、参謀計算を結び、軍全体を流れる一つの軍塊として動かす。これが弾性軍制である。

プレイヤーはこの軍制を採用した国家の最高司令部を扱う。戦闘では軍塊の形を変え、成長画面では軍制そのものを育て、戦略マップでは内政・外交・補給・地形を次の戦闘条件へ変換する。

## スライム的挙動の扱い

「スライム」は世界内の種族や兵器ではなく、挙動を説明するための開発上の比喩である。

画面上で見えるものは、人間の軍勢である。ただし、兵士の集団が次のようなスライム的挙動を持つ。

- 戦線全体が伸びる、縮む、曲がる
- 密度が高い場所ほど突破力が上がる
- 広がるほど包囲力が上がるが、結束が薄くなる
- 局所応力が高まると戦線が割れる
- 命令は即時ではなく、軍塊全体へ伝播する
- 疲労、士気、結束、兵站が形状維持に影響する

ユーザー向けの世界観では「スライムそのもの」と言わない。代わりに「軍塊」「弾性軍制」「連結隊列」「戦線密度」「応力」「結束線」などの言葉を使う。

## プレイヤーの立場

プレイヤーは一兵士でも魔物使いでもない。国家の司令部である。

操作しているもの:

- 戦闘中: 一つの軍塊として振る舞う人間の軍勢
- 成長画面: 軍制、教範、訓練、工兵、参謀制度
- 戦略マップ: 領域、補給、外交、前線圧力、戦争目的
- 内政: 軍制改革、兵站整備、工兵学校、軍医制度、徴募制度
- 外交: 通行権、補給協定、停戦、威圧、同盟、裏切り

## 画面別トーン

### 戦闘画面

戦場で軍塊を直接動かす画面。

言葉は短く、軍事的にする。

- 良い: 自軍、敵軍、軍塊、兵力、士気、結束、疲労、応力、包囲、突破、戦線、右翼、左翼
- 避ける: スライム、粘液、群体、胞子、魔法、生体器官

### 成長画面

魔法スキルではなく、国家が整える軍制の成長画面。

スキルは「人間の制度や訓練」が先で、結果として流体的挙動が強くなる形にする。

- 良い: 隊列規律、戦線管制班、突撃工兵隊、包囲測量局、兵站縫合班、軍医回収線、信号参謀、予備隊運用
- 避ける: 触手、胞子、膜、粘液、核、吸収、分裂魔法

### 戦略マップ

戦争を始める理由と、戦闘条件を作る画面。

ここでは「国家」を強く出す。資源や地形は、次の戦闘で軍塊の状態へ変換される。

- 良い: 補給路、前線圧力、徴募、工廠、通行権、停戦、威圧、同盟信用、旧要塞、湿地戦線
- 避ける: 群体繁殖、粘液領域、胞子散布など、生物群体に見える言葉

### 内政画面

施設ツリーではなく「国家がどの軍制を整えるか」を見せる。

- 良い: 軍学校、工兵庁、兵站局、軍医局、参謀本部、徴募改革、通信隊、測量隊
- 効果例: 命令遅延低下、疲労回復、結束維持、包囲精度、突破時の損耗軽減

### 外交画面

会話劇よりも「どの戦闘条件が変わるか」を重視する。

- 良い: 通行権、補給協定、共同戦線、捕虜交換、威圧、停戦期限、信用
- 効果例: 特定地域への移動解禁、補給ボーナス、援軍到着、敵士気低下、裏切りリスク

## スキル設計ルール

スキル名は「人間の組織・訓練・教範」を先に出す。
説明文では、その制度が戦場でどのスライム的挙動に変換されるかを後から説明する。

例:

- 隊列規律: 兵士が間隔と交代位置を維持する。結果として広がっても結束が落ちにくい。
- 戦線管制班: 前哨線と信号兵が側面警戒を標準化する。結果として包囲時の命令遅延が減る。
- 突撃工兵隊: 盾兵、架橋職人、爆薬係を突破線に常備する。結果として密集突破の失敗損耗が減る。
- 包囲測量局: 測量士と地図係が敵の退路を読む。結果として包囲円の閉じ方が安定する。
- 軍医回収線: 後送路と野戦病院を前線に追従させる。結果として分裂・敗走後の立て直しが速い。
- 信号参謀: 旗、太鼓、伝令、予備命令を統合する。結果として命令が軍塊全体へ伝わりやすい。

## ビジュアル方針

生成画像や背景は「人間国家の軍事画面」に寄せる。

良いモチーフ:

- 参謀本部、軍議室、地図卓、測量器具
- 工兵学校、兵站倉庫、野戦病院、徴募所
- 旗、信号灯、伝令、隊列図、戦線図
- 密集隊形、盾列、包囲線、前線圧力の図解

避けるモチーフ:

- 可愛いスライム、魔物、粘液生物
- 触手、胞子、膜、核などの生物器官
- 魔法陣、召喚、ファンタジー魔法スキルに見える絵

ただし、UIエフェクトとして流線、密度ヒートマップ、結束線、応力のひび割れを使うのはよい。これは生物表現ではなく、戦場解析の可視化として扱う。

## 判断基準

迷ったら次の順で判断する。

1. これは人間国家の軍制・兵站・外交として説明できるか
2. 戦場では軍塊の形、密度、結束、応力、命令遅延へ変換できるか
3. 表示名だけを見た時に、魔物育成ゲームに見えないか
4. プレイヤーが「この制度を育てると戦場でこう流れる」と想像できるか

この4つを満たすなら、このゲームの世界観に合っている。
