# 表示・記号の読み方

このゲームの表示は、正解の指示を出すためのナビではなく、「今どこに戦術的な材料があるか」を読むための情報です。

プレイヤーが決めるべきことは大きく2つです。

- どこを包み込むか
- どこを貫くか

その判断を助けるために、HUD、戦場上の短い記号、警告、応力表示を出しています。

## 戦場上の短い記号

敵の周辺に出る `薄`、`側`、`圧`、`包` は、攻撃指示そのものではありません。現在の形状と接触状態から見て、「そこに注目すると判断しやすい」という材料です。

| 表示 | 意味 | どう読むか |
| --- | --- | --- |
| `薄` | 中央が薄い、または隙間が出ている | 突破候補。自軍の突破力が高い時は貫く狙いになる |
| `側` | 側面へ回り込む余地がある | 包囲候補。ピンチアウトや片翼前進で腕を伸ばす狙いになる |
| `圧` | 接触圧が集中し、敵の結束が落ちやすい | すでに押せている場所。継続圧力か突破を考える |
| `包` | 包囲が進行中 | 包囲を維持すると敵の士気回復や指揮が弱る |

数字の `%` は成功率ではなく、材料の強さです。たとえば `側 72%` は「この側面は包囲材料としてかなり良い」という意味で、必ず包囲が成功するという意味ではありません。

## HUDの主な数値

| 表示 | 意味 | 上がるとどうなるか |
| --- | --- | --- |
| 士気 | 戦意。敗走判定と戦闘力に効く | 高いほど押し合いに耐え、低いと敗走しやすい |
| 結束 | スライム内部のまとまり | 高いほど形が崩れにくく、ZOCやリンクが安定する |
| 密度 | どれだけ詰まっているか | 高いほど突破圧は出るが、過密と疲労が増える |
| 幅 | 戦線の横幅 | 広いほど包囲しやすいが、薄くなりやすい |
| 疲労 | 長時間の圧力や過密で溜まる消耗 | 高いほど戦闘力と応答が落ちる |
| 圧力 | 敵と押し合っている強さ | 高いほど接触面で変形や応力が起きている |
| 隙間 | `gapRisk`。戦線の薄さ、穴の開きやすさ | 高いほど突破されやすい |
| 過密 | `crowding`。詰まりすぎ | 高いほど疲労と結束低下が進む |
| 翼 L/R | 左右の翼をどれだけ前へ出しているか | 片翼前進や包囲前進の偏りを見る |
| 接続 | リンク全体の健全性 | 低いほど分裂や形状崩壊に近い |
| 応力 / 靱性 | 局所負荷と耐えられる限界 | 応力が靱性を超えると損傷が進む |
| 亀裂 | 分裂につながる損傷の連結度 | 高いほど分裂が近い |
| 包囲力 | 現在の形が包囲に向く度合い | 高いほど敵の側面や退路を圧迫しやすい |
| 突破力 | 現在の形が突破に向く度合い | 高いほど薄い正面を貫きやすい |

## 黄色い警告

黄色い警告は「すぐ負ける」という意味ではなく、戦況の読みどころです。

| 警告 | 意味 | 対応の例 |
| --- | --- | --- |
| 戦線の隙間 | 広がりすぎ、または中央が薄い | 少し密集する、向きを変える、中央を敵から外す |
| 過密継続 | 密集を維持しすぎている | 休ませる、幅を戻す、突破を短時間で決める |
| 包囲線が薄い | 包んでいるが腕が薄い | 片翼だけ伸ばしすぎない、中央を支える |
| 包囲されています | 敵に退路を狭められている | 一点突破、回転、密集して退路を作る |
| 包囲圧 | 敵への包囲効果が進んでいる | 無理に貫かず、包囲を維持して消耗させる |
| 包囲効果 | 敵の士気回復や指揮が弱り始めている | 腕を切られないように維持する |
| 局所応力が靱性を超過 | あるリンク周辺の負荷が限界を超えている | その場所の圧力を抜く、姿勢を変える |
| 亀裂が連結しています | 損傷が線になり、分裂に近い | 圧力方向から逃がす、密集し直す |

## 応力表示

左下の `応力` ボタンを押すと、危険リンクのノード番号、負荷率、耐久度、主因が表示されます。

- 青緑の細線: 低負荷
- 黄: 注意
- 橙: 高負荷
- 点滅する赤: 現在の靱性を超過
- 線の欠け: 累積損傷
- 短い矢印: 敵圧を受けている方向
- 点線: 分裂しそうな面の予測

リンク損傷の主因は、幅そのものではなく、敵ZOCや接触面から受ける局所圧力です。展開して薄くなっている時は同じ敵圧でも耐えにくくなるため、結果的に損傷が進みやすくなります。

## 読み方の例

- `薄` が強く、自軍の突破力も高い  
  → ピンチインしながら前進して、そこを短時間で貫く。

- `側` が強く、自軍の包囲力が高い  
  → ピンチアウトしながら前進、または片翼前進で腕を回す。

- `圧` が強く、敵の結束が下がっている  
  → その接触面を維持するか、突破に切り替える。

- `包` が強く、敵の包囲値が上がっている  
  → 無理に中央突破せず、腕を切られないように包囲を維持する。

大事なのは、「包むか、貫くか」を毎回選ぶことです。密集は強いですが長く続けると疲労と過密で悪化します。展開は包囲を作れますが、薄い腕を破られる危険があります。
